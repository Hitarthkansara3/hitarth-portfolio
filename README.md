# Hitarth Kansara Premium Portfolio

## Install
npm install

## Run locally
npm run dev

## Build for production
npm run build

## Preview production build
npm run preview

## Deploy on Vercel
1. Push this project to GitHub.
2. Go to Vercel and import the GitHub repository.
3. Framework preset: Vite.
4. Build command: npm run build
5. Output directory: dist
6. Click Deploy.

## Edit content later
- Update personal/project/skill/contact data in `src/data/content.js`.
- Replace resume file in `public/resume-hitarth-kansara.pdf`.
- Update SEO tags in `index.html`.
- Tweak design tokens and styles in `src/index.css` and `tailwind.config.js`.
